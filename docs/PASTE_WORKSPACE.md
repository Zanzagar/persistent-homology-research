## Betti Numbers (paragraph form)

The homology groups are quantified by Betti numbers — the rank of each homology group, denoted βₖ. Each Betti number counts a different dimension of topological feature. β₀ counts connected components: in a geological context, these are isolated channel bodies or separate sand bodies. β₁ counts loops, or one-dimensional holes: these correspond to channel network loops, oxbow lakes, and the braiding patterns that distinguish one fluvial architecture from another. β₂ counts enclosed voids: three-dimensional cavities relevant to pore-space analysis. For our research, β₁ is the critical Betti number — a braided system has many independent loops (high β₁), while a meandering system has few (low β₁). This is the topological difference that variograms miss entirely and that MPS captures only implicitly through template reproduction, without extracting it as a formal, stable descriptor.
