## Document 2: PersistentHomology.pdf (Kemme & Agyingi, 2025)

### Paper Overview
Pedagogical introduction to PH with biological applications (3-1 supercoiled DNA). ArXiv:2505.06583v1. Designed for readers without algebraic topology background.

### Key Mathematical Definitions (verbatim notation)

**Topological Space** (Def 1): $(X, \tau)$ where $\tau$ satisfies: (1) $\emptyset, X \in \tau$; (2) closed under finite intersections; (3) closed under arbitrary unions.

**Homeomorphism** (Def 2): $f: X \to Y$ that is bijective, continuous, with continuous inverse.

**Homotopy** (Def 3): Continuous map $H: X \times [0,1] \to Y$ with $H(x,0)=f(x)$, $H(x,1)=g(x)$.

**Metric Space** (Def 5): $(M, d)$ with identity, positivity, symmetry, triangle inequality.

**Abstract Simplicial Complex** (Def 6): Collection $K$ of non-empty subsets of vertex set $V$ where: (1) every vertex $\{v\}$ is in $K$; (2) if $\sigma \in K$, all subsets of $\sigma$ are in $K$.

**Simplex construction from point clouds**:
- 0-simplex = point
- 1-simplex = edge (two points within threshold distance)
- 2-simplex = filled triangle (three pairwise-connected points)
- 3-simplex = tetrahedron (four fully connected points)
- d-simplex = convex hull of $d+1$ vertices

**Chain Group** (Def 8): For dimension $k$, the vector space $C_k(K)$ over field $\mathbb{Z}_2$, generated by $k$-simplices of $K$.

**Boundary Map** (Def 9): $\delta_k^K: C_k(K) \to C_{k-1}(K)$. Over $\mathbb{Z}/2\mathbb{Z}$: $\delta_k^K(\sigma) = \sum_{i=0}^{k} \sigma_{-i}$

**Chain Complex** (Def 11): Sequence of chain groups connected by boundary maps where $d_{k-1} \circ d_k = 0$:
$$0 \to C_k(K) \xrightarrow{\delta_k} C_{k-1}(K) \to \cdots \xrightarrow{\delta_1} C_0(K) \to 0$$

**Homology Group** (Def 12): $H_k(K) = \ker(\delta_k) / \operatorname{im}(\delta_{k+1})$
- Captures k-dimensional features that are cycles but not boundaries ("holes" in dimension k)

**Betti Numbers**:
- $\beta_0$: connected components
- $\beta_1$: loops / 1-dimensional holes
- $\beta_2$: voids / enclosed surfaces
- Examples: Circle ($\beta_0=1, \beta_1=1, \beta_2=0$), Sphere ($\beta_0=1, \beta_1=0, \beta_2=1$), Torus ($\beta_0=1, \beta_1=2, \beta_2=1$)

### Persistent Homology (Section 4)

**Filtration**: Nested sequence of simplicial complexes $K_0 \subseteq K_1 \subseteq \cdots \subseteq K_n$ built by increasing scale parameter $\epsilon$.

**Birth/Death**: Topological features appear (birth) and disappear/merge (death) during filtration. Persistence = death - birth. Long-persisting features = meaningful; short-lived = noise.

**Stability Theorem** (refs [6,7,24]): Small perturbations of input data lead to small changes in persistence diagram. Key distances: **bottleneck distance** and **Wasserstein distance**.

**Visualization**:
- **Persistence Barcodes**: Horizontal bars, length = persistence
- **Persistence Diagrams**: Points in 2D (birth on x-axis, death on y-axis). Points far from diagonal = significant features.

**Vectorization for ML**: Persistence landscapes, persistence images.

### DNA Application Results
3-1m-Supercoiled DNA: $\beta_0=1$ (single connected component), $\beta_1=3$ (three loops/cycles — biologically relevant twists/knots), $\beta_2=0$ (no voids).

## Document 3: essence_thought_experiment.pdf (Hoydic, January 2026)

### The Thought Experiment
Progressive degradation of a channelized reservoir image:
- **Variant A (Occlusion)**: Add ellipses that hide features — information preserved but inaccessible
- **Variant B (Erosion)**: Remove geological features themselves — information irreversibly destroyed
- **Core question**: At what point is "essence" lost?

### Five Problems Identified
1. **Occlusion vs erosion are fundamentally different** — Variant A preserves info (occlusion problem), Variant B destroys it. The qualia convergence framework faces an occlusion problem (sparse observations of full geology).
2. **Path dependence** — Different degradation orders yield different "minimal essence" sets. No unique answer.
3. **Feature entanglement** — Geological features are physically coupled (Leopold-Wolman: λ = 10-14W; R_c = 2-3W). Independent removal creates geologically impossible states, leaving the constrained manifold M.
4. **Observer-relativity** — Essence depends on measurement system:
   - Human sedimentologist: recognition up to ~60% occlusion
   - DINOv2: might fail at 40% or persist to 70% depending on pretraining
   - **Persistent homology (H₁ loops)**: particularly sensitive to channel connectivity — few ellipses at intersections could destroy homology
   - Variogram-based features: respond differently to sparse vs dense occlusion
5. **Minimal vs sufficient essence** — Minimality is fragile; sufficiency (redundant features) more robust for retrieval.

### Alternative Proposal: Essence as Generator Invariance
Rather than degradation thresholds, define essence through **invariance across generative processes**:

**Formal Framework**:
- Image space X, Parameter space Θ, Generator G: Θ → X
- Constrained manifold M ⊂ Θ where physical constraints C_j(θ) = 0
- Representation function f: X → Z
- **Essence(f)** = ∩_G {information about θ captured by f when trained on G}

**LOGO Test** (Leave-One-Generator-Out): Train on {G_j : j ≠ i}, test on G_i. Features that transfer = essence. Features that don't = generator artifacts.

### Why Generator Invariance > Degradation Threshold
| Property | Degradation-Based | Invariance-Based |
|----------|-------------------|------------------|
| Uniqueness | No (path-dependent) | Yes (single intersection) |
| Respects physics | No (impossible states) | Yes (generators enforce constraints) |
| Observer-independent | No | Yes (generator agreement) |
| Task-relevant | Indirect | Direct (retrieval capability) |
| Operationally testable | Arbitrary choices | LOGO test well-defined |

### Four Lenses for Essence (Convergence)
1. **Generator invariance (LOGO)** — canonical essence definition
2. **Degradation robustness (occlusion test)** — practical robustness
3. **Physical interpretability (transfer prediction)** — scientific meaning
4. **Expert agreement (validation survey)** — human judgment alignment

### Key Insight for PH Paper
PH is one of three representation pathways (classical geostatistics, learned/DINOv2, topological). The topological pathway is **sensitive to channel connectivity** — even few ellipses at intersections could change H₁ homology. Essence best understood through convergence of all pathways. The ellipse experiment = robustness test, not essence definition.

## Document 4: persistent_homology_framework_analysis.md

### Core Thesis
The three pathways (Classical, Learned/DINOv2, Topological/PH) extract features at different "dimensions" of the signal. The convergence regularization loss L_conv = Σ||f_i(x) - f_j(x)||² enforces consistency — this IS the boundary operator's coherence condition (∂_{k-1} ∘ ∂_k = 0) in a deep structural sense.

### Key Extensions Proposed

1. **Multi-Scale PH** — Standard PH uses single filtration parameter ε. Geological images have structure at multiple scales (bar spacing, channel sinuosity, floodplain texture). Multi-parameter persistence tracks topological features across scales = "objects of different dimensions crafting each other."

2. **Hyperbolic Embedding Connection** — Poincaré ball encodes hierarchy (center=general, boundary=specific). Persistence diagram IS a hierarchical decomposition: long bars = essential, short bars = noise/detail. Natural map: longest-lived H₁ features → position along specificity axis of Poincaré ball.

3. **LOGO Test + Topological Invariance** — PH is provably stable under perturbations (Chazal stability theorem). Topological features should transfer between generators better than learned features. **Testable prediction**: topological pathway degrades least under generator shift in LOGO.

4. **Topological Dictionary Learning** — Learn canonical persistence diagrams ("topological atoms") such that any image's PH decomposes as sparse combination. Atoms = canonical primitives ("sinuous channel with one oxbow", "braided network with 3-5 connections"). Links generator parameters → topological representation → essence claim.

### Practical Recommendations
- Phase 1: Test correlation between PH statistics (total H₁ persistence, long-lived H₁ count) and generator parameters (thread count, sinuosity). Strong correlation (r > 0.5) validates topology-generator connection.
- LOGO: Report per-pathway degradation. If topological < learned degradation, supports multi-pathway architecture (ADR-001).

## Documents 7-9: Qualia Convergence Framework (Vision, Architecture, Implementation)

### Complete Vision (v5.4) — Key Points for PH Paper
- **Operational definition of essence**: Invariant structure stable under (1) multiple pathways, (2) multiple modalities, (3) multiple generative processes. Epistemic, not metaphysical.
- **Pragmatic operationalism**: Following Peirce/James/Dewey — meaning = consequences. If LOGO passes, retrieval works, experts agree → justified in calling it "essence."
- **"Qualia" is metaphorical** — NOT phenomenological qualia. Signals ambition beyond surface statistics.
- **Three-pathway architecture**: Classical (variograms, ~50 dims), Learned (DINOv2, 768 dims), Topological (PH, 20-50 dims). Each = independent epistemic "darśana."
- **Topological pathway**: H₀ connected components, H₁ loops, vectorized via persistence entropy/landscapes/images.
- **Hyperbolic embedding**: Poincaré ball model. Center = general (e.g., "fluvial"), boundary = specific (e.g., "high-sinuosity meandering").
- **Sākṣī validation**: 10 independent witnesses including LOGO, transfer prediction, expert validation (Cohen's κ), adversarial tests.
- **Adversarial test 23.1**: Variogram-matched topology swap — tests whether system distinguishes braided vs meandering when variograms are identical. H₁ should capture the difference.

### Architecture Decisions (v1.1) — Key Points
- **ADR-001**: Three pathways chosen for philosophical grounding (Sākṣī), complementary sensitivity, failure isolation.
- **ADR-008**: LOGO as primary invariance test. More stringent than standard domain adaptation (complete exclusion, no adaptation).
- **ADR-S04**: TDA is an ablation study until H₁ hypothesis validated. If >70% accuracy → core pathway; if <60% → ablation only.
- **Open question**: Multi-scale TDA — should multi-parameter persistence be used?

### Implementation Spec (v6.3) — Key Points
- **Phase 2**: Implement topological pathway. Cubical complex via Giotto-TDA. H₀ and H₁ only (skip H₂ for 2D).
- **Decision Point 2.1**: H₁ hypothesis test — classify braided vs meandering using only H₁ features from images with matched variograms.
- **Phase 3**: LOGO validation. Degradation thresholds: <20% strong, 20-30% moderate, >30% need pivot.
- **SEDT filtration**: Signed Euclidean Distance Transform for binary facies images.
